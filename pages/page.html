Some useful shell commands for web development

  1. To create a new directory: mkdir directory_name
  2. To navigate into a directory: cd directory_name
  3. To print current working directory: pwd
  4. To create a new file: touch index.html
  5. To open the directory in VS Code: code . || open -a "Visual Studio Code" .
  6. To open 'index.html' in chrome browser: open -a "Google Chrome" index.html
